The Appointment Directory provides a simple, searchable catalogue of local services and appointment types. Designed for reception staff, it helps teams quickly find the right appointment option and understand booking rules.

---

## Overview

The directory is fully integrated into the top-level navigation and serves as a reference tool for:
- Understanding appointment types available
- Identifying which staff team handles what
- Finding booking rules and duration
- Supporting new receptionist training

---

## Appointment Types

Each appointment entry includes:

### Core Information
- **Name** — Clear, descriptive appointment type
- **Staff Team** — Which team handles this appointment
- **Duration** — Appointment length in minutes
- **Colour Coding** — Visual identification by staff team
- **Notes** — Additional guidance or rules

### Status
- **Enabled/Disabled** — Control visibility
- **Default Appointments** — System-provided templates
- **Custom Appointments** — Surgery-specific types

---

## Staff Teams

Staff teams organise appointments and provide visual grouping through colour coding.

### Dynamic Teams
Admin team members can add, edit, and manage staff team options. This allows surgeries to reflect their actual organisational structure.

### Team Features
- **Custom Labels** — Surgery-specific team names
- **Colour Coding** — Visual identification
- **Ordering** — Control display order
- **Enabled/Disabled** — Show or hide teams

### Built-in Teams
Common team types are available by default:
- Practice Nurse (PN)
- Healthcare Assistant (HCA)
- Doctor (Dr)
- All Staff
- And more...

Surgeries can modify, add, or remove teams to match their structure.

---

## CSV Import

Rapid bulk updates are possible through CSV import functionality.

### Import Process
1. Prepare CSV file with appointment data
2. Upload via admin interface
3. System validates and processes
4. Appointments created or updated

### CSV Format
- Headers: Name, Staff Team, Duration, Notes, etc.
- One row per appointment type
- Validation ensures data quality
- Errors reported for correction

### Use Cases
- Initial setup from existing lists
- Bulk updates after reorganisation
- Regular maintenance updates
- Importing from other systems

---

## Filtering

The directory supports multiple filtering options to help staff find appointments quickly.

### Filter Types

1. **Search**
   - Text search across names and notes
   - Instant results as you type
   - Highlights matching terms

2. **Staff Team Filter**
   - Filter by specific staff team
   - See all appointments for a team
   - Useful for team-specific workflows

3. **Status Filter**
   - Show enabled only (default)
   - Include disabled for admin
   - Quick visibility toggle

### Filter Features
- Combine multiple filters
- Clear filters easily
- Persistent preferences
- Fast performance
